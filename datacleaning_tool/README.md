## 🚀 Quick Start

### Installation
```bash
# Install dependencies
pip install -r requirements.txt

# Verify installation
python -c "import enhanced___init__ as cortex; cortex.check_installation()"

# Launch the platform
streamlit run enhanced_app.py
```

### Docker Deployment
```bash
# Build and run with Docker
docker build -t cortexx-platform .
docker run -p 8501:8501 cortexx-platform
```

### First Use
1. Open your browser to `http://localhost:8501`
2. Upload your sales or business data
3. Use automated data quality assessment
4. Generate executive dashboards
5. Export professional reports
